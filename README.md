# Storm-Avoidance: 降る雨を避けるミニゲーム


## 概要

本プロジェクトは、スマートフォンの加速度センサーを使用して画面上のキャラクターを操作し、上から降ってくる雨粒を避けてスコアを競うシンプルなAndroidゲームです。Firebase Remote ConfigやAnalyticsなどのモダンな機能も組み込まれています。


## 主な機能

* **加速度センサー操作**: デバイスを左右に傾けてキャラクターを操作します。
* **難易度選択**: Easy、Normal、Hardの3段階の難易度を選択可能です。各難易度の設定はFirebase Remote Configで動的に制御されます。
* **ハイスコア保存**: スコアは端末内に永続的に保存され、ハイスコアとして記録されます。
* **音量・振動設定**: 設定画面からBGM/SEの音量調整、およびゲームオーバー時の振動ON/OFFを設定できます。
* **初回チュートリアル**: 初回起動時のみ操作方法に関するチュートリアルが表示されます。
* **ゲーム一時停止**: ゲーム中に一時停止/再開が可能です。
* **Firebase連携**:
    * **Remote Config**: 難易度ごとのパラメータ（雨粒の発生頻度、スコア倍率）を外部から設定・変更可能。
    * **Analytics**: プレイ開始、プレイ時間、最終スコア、レベルアップなどの主要なイベントを記録・分析。

## インストールとセットアップ

### 1. プロジェクトのクローン

```bash
git clone https://github.com/oteg2app2/acc-02.git
cd acc-02

### 2. Firebase設定（必須）

プロジェクトをビルド・実行する前に、Firebaseコンソールでプロジェクトを設定し、以下の手順を完了してください。

1.  FirebaseコンソールでAndroidプロジェクトを作成し、パッケージ名 `com.example.acc_02` を登録します。
2.  生成された `google-services.json` ファイルを `app/` ディレクトリに配置します。（注: `google-services.json` はセキュリティのためGitHubにはアップロードしないでください）

### 3. Remote Configのパラメータ設定

本アプリの難易度パラメータはRemote Configに依存します。Firebaseコンソールで以下のキーと値を設定してください。

| キー | デフォルト値（型：数値） | 説明 |
| :--- | :--- | :--- |
| `easy_spawn_chance` | 1 | Easyモードでの雨粒発生確率（低） |
| `easy_multiplier` | 1 | Easyモードでのスコア倍率 |
| `normal_spawn_chance` | 3 | Normalモードでの雨粒発生確率（中） |
| `normal_multiplier` | 3 | Normalモードでのスコア倍率 |
| `hard_spawn_chance` | 6 | Hardモードでの雨粒発生確率（高） |
| `hard_multiplier` | 5 | Hardモードでのスコア倍率 |

### 4. ビルドと実行

Android Studioでプロジェクトを開き、実機またはエミュレータで実行してください。

## 使い方（操作方法）

1.  **初回起動**: チュートリアル画面が表示されます。「START」ボタンを押して進みます。
2.  **レベル選択**: Easy、Normal、Hardのいずれかを選択するとゲームが開始します。
3.  **キャラクター操作**: スマートフォンを**左右に傾ける**と、キャラクターが水平方向に移動します。
4.  **目的**: 上から降ってくる雨粒に当たらないように避け続けます。
5.  **スコア**: 雨粒を避けるたびにスコアが加算されます。
6.  **レベルアップ**: 一定スコアに達するとレベルアップし、ゲームが加速します。
7.  **ゲームオーバー**: 雨粒に当たるとゲームオーバーです。
